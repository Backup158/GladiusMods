# The Three Textures
It's best to work on diffuse first, then import it to the others as a guide.

Say I drew goggles on the diffuse and exported. I can then import that dds onto the SIC, and set the diffuse layer to 50% opacity. Now, I have a 1:1 reference for what to draw on the SIC (and same idea for the normals)
## Diffuse
What you normally think of as texturing. Just the base colors.
### Coloring Existing Colors
For the Jopall Indentured Squadrons, I wanted to color the tunic turquoise, but I didn't want to have to redo all the shadows already done on it.
1. Import base diffuse
2. Create new layer
3.  With an opacity 80% brush, make broad strokes over the areas you want to change
4. Optionally, add patterns. I used the splatter brush to make camo splotches
5. Use a 100% eraser to erase the parts you don't want to change (belts, edges of armors, etc)
6. save
7. merge layers and export
8. close WITHOUT saving
