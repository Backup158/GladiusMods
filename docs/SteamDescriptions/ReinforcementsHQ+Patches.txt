[h1]Load Order[/h1]
You don't have to [b]use[/b] all of them, but you need to [b]install[/b] all of them. 

From testing, that seems to be what's necessary. If it turns out that isn't necessary, great!
You'll know you need to do this if you crash on startup. Oh yeah, how to do this:
Subscribe to all of them, then disable the ones you won't use. Then, it'll only patch the ones you have enabled; it will put some messages in the game log for the missing factions, but you can ignore them.

Exact order shouldn't matter, but I have these above (visually higher) the faction mods and HQ+.
[code]
HQ+Core Unit for Reinforcements Series Factions
Reinforcements - World Eaters
HQ+Core Unit Production
[/code]

[b]Why Not the Titans?[/b]
Since the HQ is a unit, it's not as simple as adding the production to it. I'd have to create a custom ability based on the regular infantry ability, which while not difficult, would require more testing to make sure it works and won't crash if you didn't install [i]Reinforcements - Collegia Titanica[/i].
If that condition is satisfied, I'll update it here. Otherwise, it'll go into a separate mod (if I even make it since I don't even play much Gladius nowadays .-.)
